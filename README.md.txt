# Pet Shop Management System

## Project Overview
This is a team-based Pet Shop Management System website project.

My contribution was Database Design and Management using MySQL.

## Role
Database Developer

## Responsibilities
- Designed database schema
- Created tables and relationships
- Inserted sample data
- Managed sales and inventory records
- Developed SQL queries for reporting

## Technologies Used
- MySQL
- MySQL Workbench
- SQL

## Database Tables

1. Admin
2. Customers
3. Animal
4. Bird
5. Product
6. Food
7. Sales
8. Sales Details

## Features

- Pet Inventory Management
- Product Management
- Food Inventory Tracking
- Sales Management
- Revenue Analysis
- Customer Data Management

## Sample SQL Reports

### Total Revenue

```sql
SELECT SUM(Amount) AS Total_Revenue
FROM sales;

SELECT AVG(Price) AS Avg_Product_Price
FROM product;

SELECT *
FROM animal
ORDER BY Price DESC
LIMIT 1;

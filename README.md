# Pet Shop Database Management

## Project Overview

This project is part of the **Pet Shop Management System (Team Project)** developed for academic purposes.

My contribution focused on **Database Management** using MySQL, including database design, table creation, data insertion, and SQL query development.

---

## My Role

**Database Management Developer**

### Responsibilities

* Designed the database schema for the Pet Shop Management System.
* Created tables for Admin, Customers, Animals, Birds, Products, Food, Sales, and Sales Details.
* Inserted and managed sample data.
* Developed SQL queries for reporting and analysis.
* Generated sales and revenue reports.
* Maintained database structure and integrity.
